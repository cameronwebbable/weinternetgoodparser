Requirements
===
- Ruby 2.7.2
- Bundler (`gem install bundler`)
	
How to Run
===
- Open terminal
- cd into the directory for this repository
- If you haven't, run `bundle install`
- To run script: `bundle exec ruby parse2.rb`
- On being prompted, enter week you'd like to get stats for
